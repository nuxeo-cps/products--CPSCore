CPSCore

  This product declares the key framework services for the Nuxeo Collaborative
  Portal Server (CPS) v3 (or CPS3). 

  You can find more information about CPS3 in the 
  "EuroPython slides":http://www.europython.org/Talks/Slides/cps_europython_2003.pdf
  
  See also the "Nuxeo web site":http://www.nuxeo.com/en/ and
  "CPS Project":http:/www.cps-project.org/ for more details.

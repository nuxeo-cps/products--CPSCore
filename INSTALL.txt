This product declares only the key framework services for the Nuxeo
Collaborative Portal Server (CPS) v3 (or CPS3).

This package should come along with a distribution of CPS3. Either one
of the official CPS3 releases or from a svn checkout.

If it comes along a CPS3 release then check the instructions included
within the release of CPS3 for further information about the
installation.

See also http://svn.nuxeo.org/trac/pub/file/CPSRelease/ for release
information

If you're looking for support goto http://www.cps-project.org/
